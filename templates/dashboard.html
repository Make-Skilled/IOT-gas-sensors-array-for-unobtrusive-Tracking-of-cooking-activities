Project Overview

About

Smart Kitchen Monitoring System is an IoT-based solution that helps you manage and monitor your cooking activities efficiently. It combines hardware sensors with a user-friendly web interface for a seamless cooking experience.

Key Features

  • Real-time temperature monitoring
  • Gas leak detection system
  • Recipe management
  • Step-by-step cooking guides
  • Activity tracking
  • Safety alerts and notifications

Hardware Components

  • Arduino Microcontroller
  • Temperature Sensor (DS18B20)
  • Gas Sensor (MQ-2)
  • LCD Display
  • Keypad Interface
  • Servo Motor
